Basis

A set of vectors that can be combined to cover the entire space (in our case, a 2d plane). These vectors also need to be efficient, in the sense that you shouldn't be able to form one vector from any of the others.
Linear Independent :

All other vectors can't be used to form the other vector. In a 2d Space you can't have more than 2 vectors represent that space. You can always form that third vector with the first two.

The Span Requirement :

In a 2d, the basis doesn't need to be orthogonal (or 90 deg of each other)! Imagine a vector on the x axis pulling any other vector along the x axis, you will cover the entire plane.

Rank

Number of dimensions in the output of a transformation. Here we are reducing the number of dimensions from 2 to 1. Therefore our Rank is 1.

Kernel (or Null Space)

The Kernel of a linear transformation (red points) is the space that goes to 0 during the linear transformation.
When no dimensionality reduction happens eg. 2d to 1d, the only point in the kernel will be 0. 0 is always in the kernel.
